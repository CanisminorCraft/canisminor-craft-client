#### 安装说明

- 安装 *svn.msi*
- 新建文件夹,并右键 *SVN Checkout*
- URL处填写:

```bash
svn+ssh://svn@svn.coding.net/canisminor1990/cmc-client/trunk
# 等待下载结束,使用`cmc-launcher.jar`启动
# PS: 环境 ≥ Java7; 推荐使用Java 8

```

<br>

#### 更新说明

- 关注官网News下 *Client* 动态
- 右键 *SVN Update* 更新客户端
